Code in C++ language that can find minimum and maximum number present in an array.

Sample code

            
                #include < bits/stdc++.h >
                using namespace std;

                typedef long long ll;

                int main(){
                    int n;
                    cout<<"Enter the size of an array: ";
                    cin>>n;
                    vector < int > v(n);

                    for(int i=0;i < n; i++){
                        cin>>v[i];
                    }

                    sort(v.begin(),v.end());

                    cout << " Maximum element: " << v[n-1] << "\n";
                    cout << "Minimum element: " << v[0] << "\n";

                    return 0;
                }
            
            
                
Sample Input:

                Enter the size of an element: 5 
                13 24 14 56 1
                
Sample Output:

                Maximum element: 56
                Minimum element: 1

Variables Uesd in the program:

n as the total number of elements.
v as the vector.

Time Complexity:
We are using inbuilt sort function.

  • Time complexity in Best Case: O(n * log n).
    Where n=size of vector


    • Space Complexity:
    We are using vector for storing value O(n).
    Where n=size of vector
